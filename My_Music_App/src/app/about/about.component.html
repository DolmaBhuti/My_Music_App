Welcome To My First Angular Web App

Welcome to my Angular-based music discovery platform, where I connect users to a world of music using the Spotify API. My app offers an immersive experience for exploring new releases, discovering albums, and artists seamlessly.

At the core of my application is a custom authorization Node API that prioritizes top-tier security and functionality. I've developed this API to ensure the utmost protection of user data, leveraging Redis server for JWT refresh tokens for a robust authentication system.

Additionally, I've utilized MongoDB for efficient management of user data and preferences. Users' favorite tracks are securely stored in my MongoDB database, providing a personalized experience each time the app is used.

Check out the github page at https://github.com/DolmaBhuti/My_Music_App.
